# 搭讪脚本

## 号码收场

1. 说“哈喽”
2. 若对方被吓到了，则说“吓到你了”；若对方感到紧张，则说“你有点紧张”。可选寒暄
3. 说“我刚刚在那边看到你了”
4. 说“想跟你认识一下”。可选闲聊
5. 说“加个微信可以吗”
6. 若对方犹豫不决，则说“来嘛”；若对方问为什么，则说“就是交个朋友”。可选告别

## 亲吻收场

1. 说“哈喽”
2. 若对方被吓到了，则说“吓到你了”；若对方感到紧张，则说“你有点紧张”。可选寒暄
3. 说“我刚刚在那边看到你了”
4. 说“想跟你认识一下”。可选闲聊
5. 心电感应测试（DHV）
6. 说“送你个礼物”，手抬起虚握，等对方接的时候转为牵手（IOI）
7. 说“好看的皮囊千篇一律，有趣的灵魂万里挑一，你是个有趣的人吗”，等对方回答是（IOI）
8. 沉默，等对方问所以呢（IOI）
9. 说“亲我一口可以吗”（CT）
10. 若对方犹豫不决，则说“来嘛”；若对方问为什么，则说“证明你的有趣”。可选告别
