# 搭讪脚本

1. 开场
    1. 中性开场：问候（说“哈喽”“嗨”“你好”之类的话），若对方注视着你则视为一个破绽
    2. 共情开场：若对方被吓到了则说“吓到你了”，若对方感到紧张则说“感觉你有点紧张啊”
    3. 情境开场：寒暄（说“你是要回家还是怎么样”“你们是在逛街吗”“你是一个人来的吗”“你是在找人吗”“你是在打电话吗”“你是在等朋友吗”“你觉得这家店的氛围怎么样”“你们经常来这里吗”“咖啡好喝吗”“你在看什么书呀”之类的话），若对方额外回答或主动提问则视为一个破绽
    4. 二次开场：若对方想要离开则说“等一下”，若对方不听你讲完便拒绝则说“听我把话讲完”
    5. 经典开场：说“我刚刚在那边看到你了，想跟你认识一下”
2. 中场
    1. 礼仪中场：说“我叫xxx”，然后尝试握手，若对方跟你握手则视为一个破绽
    2. 称呼中场：若对方没有说自己叫什么则说“你呢”
    3. 地区中场：说“你是哪里的”，若对方额外回答或主动提问则视为一个破绽
    4. 年龄中场：说“你多大了”，若对方额外回答或主动提问则视为一个破绽
    5. 配偶中场：说“你有男朋友了吗”，若对方回答没有则视为一个破绽否则说“那你想换一个吗”，若对方回答想换则视为一个破绽
3. 收场
    1. 游戏收场：和对方玩一个游戏（如心电感应测试：让对方在心里想一个一到十之间的数字，你来猜，注意这里猜7的命中率稍大），玩之前先介绍一下这个游戏（如心电感应测试：说“有一个测试可以看出我们之间是不是心有灵犀，要不要玩一下”），若你赢了游戏则视为一个破绽
    2. 牵手收场：消耗三个破绽，说“送你个礼物”，手抬起虚握，等对方接的时候转为牵手，若对方让你牵着则视为一个破绽
    3. 亲吻收场：建立在牵手收场成功的基础之上，说“好看的皮囊千篇一律，有趣的灵魂万里挑一，你是个有趣的女生吗”，若对方回答是的则视为一个破绽，然后沉默，若对方问所以呢则视为一个破绽，然后消耗三个破绽，说“那你愿意亲我一口吗”
    4. 约会收场：说“你现在赶不赶时间”，若对方不赶时间则说“那我们一起去喝点东西吧”，若对方愿意一起则与对方即时约会
    5. 号码收场：说“加个微信可以吗”，若对方犹豫不决则说“来嘛”，若对方问为什么则说“就是交个朋友”，若对方加了微信则说“有机会约你出来喝杯咖啡”
