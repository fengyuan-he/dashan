# 搭讪脚本

1. **中性开场：说“哈喽”**
2. **共情开场：若对方被吓到了，则说“吓到你了”**
3. 情境开场：寒暄（说“你是要回家还是怎么样”“你们是在逛街吗”“你是一个人来的吗”“你是在找人吗”“你是在打电话吗”“你是在等朋友吗”“你觉得这家店的氛围怎么样”“你们经常来这里吗”“咖啡好喝吗”“你在看什么书呀”之类的），并尝试闲聊，若闲聊成功，则跳至13.挤占收场
4. 二次开场：若对方想要离开，则说“等一下”
5. **经典开场：说“我刚刚在那边看到你了”**
6. **标准开场：说“想跟你认识一下”**
7. 握手中场：说“我叫xxx”，并尝试握手
8. 称呼中场：若对方没有说自己叫什么，则说“你呢”
9. 地区中场：说“你是哪里的”，并尝试闲聊，若闲聊成功，则跳至13.挤占收场
10. 年龄中场：说“你多大了”，并尝试闲聊，若闲聊成功，则跳至13.挤占收场
11. 配偶中场：说“你有男朋友了吗”
12. 需求中场：若对方回答没有，则说“那你想找一个吗”，否则说“那你想换一个吗”
13. 挤占收场：说“你现在赶不赶时间”，若对方要赶时间，则跳至17.微信收场
14. 牵手收场：说“那我们一起去喝点东西吧”，并尝试牵手
15. 同行收场：若对方不愿一起，则说“走嘛”，若对方还是不愿，则跳至17.微信收场
16. 约会收场：即时约会
17. **微信收场：说“加个微信吧”**
18. **朋友收场：若对方问为什么，则说“就是交个朋友”**
